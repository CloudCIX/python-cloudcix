
**Basic Usage**: ``cloudcix.api.NLP.<service>.<method>``

Services
--------

Embedding USE
^^^^^^^^^^^^^^
This service is used in the NLP application to contact a NN Model and return the embeddings for the sent text.
**Basic Usage**: ``cloudcix.api.NLP.embedding_use.<method>``

Available Methods
    - ``cloudcix.api.NLP.embedding_use.create``  
        -This method creates a list of embeddings from the sent list of text in request.data
